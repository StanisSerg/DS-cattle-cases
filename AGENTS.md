# AGENTS.md

> **Для Kimi Code:** этот файл читается автоматически. Не редактируй вручную.

## Правила для агентов

1. **Создание кейса:** используй `bash scripts/case-new.sh CASE-NNN farm-slug [category]`. Не создавай папки вручную.
2. **Валидация:** перед любым коммитом запускай `bash scripts/case-validate.sh --all`.
3. **Структура:** каждый кейс должен содержать `CASE-NNN-slug.md` и папки `raw/`, `charts/`, `reports/`, `scripts/` (даже если пустые).
4. **Сырые данные:** исходные таблицы и PDF — только в `raw/`, не в корне кейса.
5. **Ссылки:** проверяй, что markdown-ссылки на `raw/` и `charts/` разрешаются.
6. **Графики:** предпочитай скрипты в `scripts/generate_charts.py`; PNG коммить, но имейте источник.
7. **Cross-repo:** при ссылке на Pack/operations используй относительные пути от корня IWE.
